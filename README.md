ipToHost
===================
[![License](http://img.shields.io/:license-mit-blue.svg)](http://doge.mit-license.org)

## Purpose
Used to take a single ip or file with ip addresses and get their
hostname(s).

## Getting Started
```
bash install.sh
```

## Usage
```
ipToHost /root/hosts.txt
ipToHost hosts.txt
ipToHost 216.58.217.46
```
